# DeweyWiki [https://nothatkind.github.io/DeweyWiki/]
Chooses a random **Dewey Decimal category** and shows a **starred, featured, or highly rated** Wikipedia article from the category.

Simply use **Shuffle** to get a new page and **Share** to share. No other buttons!

Mobile-friendly design.

You can save the HTML file locally as well. Works standalone and leaves no tracks. *However, it may not run standalone on iOS because of protections I have not worked on yet.*

Currently has a Tron theme. I may add others. In any case you can theme it using your AI friend or by directly editing the style elements.

While I did spend time on some "real" coding and troubleshooting here, **the development was done by Claude.** I have sanitised it and no information of yours is made available to Anthropic.

Feel free to fork, add features, and **tell me about them.**

> The Dewey Decimal Classification (DDC) system, developed by Melvil Dewey in 1876, is a widely used, structured library classification system that organizes non-fiction books by subject using decimal numbers. It divides knowledge into 10 main classes, ranging from 000 to 900, with further subdivisions for specific topics.
